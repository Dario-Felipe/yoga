## overflow

```javascript state
const Square = styled.p`
  width: 200px;
  height: 50px;

  ${overflow}
`;

render(() => {
  return (
  <Square of="scroll">
     This is a description to show the example of using the overflow property 
     with the scroll value     
  </Square>
  );
});
```
