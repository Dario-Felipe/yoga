# Web Accessibility

Accessibility, also known as the ability to access, means that devices,
services, and products are accessible to people with disabilities. Some people
have to use assistive technologies which make these things accessible for them.

In this section you will find several tips and guides to help develop web user
interfaces with accessibility in mind.
