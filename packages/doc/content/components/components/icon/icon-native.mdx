### Reference

### Basic usage

```javascript type=expo
<>
  <Icon as={Accessibility} />
</>
```

### Usage with custom size, fill and stroke

```javascript type=expo
<>
  <Icon as={AlertCircle} size="medium" fill="energy" stroke="energy" />
</>
```

### Usage with custom width, height, fill and stroke

```javascript type=expo
<>
  <Icon as={AlertCircle} width="medium" height="medium" fill="energy" stroke="energy" />
</>
```
