### Basic usage

```javascript
<Icon as={Accessibility} />
```

### Usage with custom size, fill and stroke

```javascript
<>
  <Icon
    as={AlertCircle}
    width="medium"
    height="medium"
    fill="energy"
    stroke="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```

### Usage with system props

```javascript
<>
  <Icon
    as={AlertCircle}
    w={50}
    h={50}
    margin="medium"
    fill="energy"
    stroke="energy"
  />
  <Icon as={Home} size="medium" fill="hope" />
</>
```
