# Popover

### Reference

Gympass `<Popover />` description

### Usage

#### Default

```javascript
<Popover
  title="Make wellbeing universal!"
  description="Try moving to another spot. Your new favorite activity could be arround the corner."
>
  <Icon as={Info} size="large" />
</Popover>
```
