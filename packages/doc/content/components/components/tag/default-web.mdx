### Usage

```javascript state
render(
  <Box display="flex" flexDirection="column">
    <Tag>default</Tag>
    <Tag mv="medium" variant="success">success</Tag>
    <Tag variant="informative">informative</Tag>
    <Tag variant="attention" small>attention</Tag>
  </Box>
);
```

### Ellipsis
The tag component can recieve a ellipsis prop that will put a ellipsis (...) if is a large text.

```
<Box d="flex">
  <Tag maxWidth="138px" ellipsis >default with large text</Tag>
</Box>
```
