# BottomSheet

The component Bottom sheet notification is a variation of the Bottom sheet component and has the objective to provide maximum priority feedback for the user.

### Usage

### BottomSheet with one actions

```javascript state
render(() => {
  const [isOpen, setIsOpen] = useState(false);

  const handleOpen = () => {
    setIsOpen(true);
  };

  const handleOnClose = () => {
    setIsOpen(false);
  };

  const handleOnAction = () => {
    alert('This is your custom action');
    handleOnClose();
  };

  return (
    <>
      <Button onClick={handleOpen}>BottomSheet with one actions</Button>

      <BottomSheet isOpen={isOpen}>
        <BottomSheet.Header>We confirm your eligibility</BottomSheet.Header>
        
        <BottomSheet.Content>
          Double check your info and company selection. If it still doesn’t work,
          contact your HR team.
        </BottomSheet.Content>
        
        <BottomSheet.Footer>
          <Button onClick={handleOnAction}>Ok, I understand</Button>
        </BottomSheet.Footer>
      </BottomSheet>
    </>
  );
});
```
