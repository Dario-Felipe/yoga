# Dialog

### Reference

Gympass `<Dialog />` is the proper component to show alert messages.

The `Dialog` may have an icon, a custom action and a close button.

### Usage

### Dialog with one action

```javascript state
render(() => {
  const [isOpen, setIsOpen] = useState(false);

  const handleOpen = () => {
    setIsOpen(true);
  };

  const handleOnClose = () => {
    setIsOpen(false);
  };

  return (
    <>
      <Button onClick={handleOpen} secondary>Dialog with one action</Button>

      <Dialog
        isOpen={isOpen}
        onClose={handleOnClose}
      >       
       <Dialog.Header>
         <Text.H4 ta="center"> Title </Text.H4> 
       </Dialog.Header>
       <Dialog.Content>
         <Text color="deep"> Subtitle</Text>
       </Dialog.Content>
       <Dialog.Actions>
          <Button onClick={handleOnClose} secondary>Ok, got it</Button>
       </Dialog.Actions>
      </Dialog>
    </>
  );
});
```
