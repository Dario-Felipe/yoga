# Dialog

### Reference

Gympass `<Dialog />` is a component has the objective o provide priority feedback for the user

As it could be seen in most components, the Dialog has its own compounds:

- `<Dialog.Header />`
- `<Dialog.Content />`
- `<Dialog.Actions />`

### Usage

### Dialog with one action

```javascript state
render(() => {
  const [isOpen, setIsOpen] = useState(false);

  const handleOpen = () => {
    setIsOpen(true);
  };

  const handleOnClose = () => {
    setIsOpen(false);
  };

  return (
    <>
      <Button onClick={handleOpen}>Dialog with one action</Button>

      <Dialog
        isOpen={isOpen}
      >       
       <Dialog.Header>
         <Text.H4 ta="center"> Title </Text.H4> 
       </Dialog.Header>
       <Dialog.Content>
         <Text color="deep"> Subtitle</Text>
       </Dialog.Content>
       <Dialog.Actions>
          <Button onClick={handleOnClose} secondary>Ok, got it</Button>
       </Dialog.Actions>
      </Dialog>
    </>
  );
});
```
