### Reference

Compounded `Button` used to display visual icon elements, supporting the same states as <GatsbyLink to="/components/button/default">`<Button />`</GatsbyLink>

### Usage

#### Default

```javascript
<Box display="flex" justifyContent="space-evenly" width="100%">
  <Button.Icon icon={Booking} />
  <Button.Icon icon={Booking} secondary />
</Box>
```

#### With large

```javascript
<Box display="flex" justifyContent="space-evenly" width="100%">
  <Button.Icon icon={Booking} large/>
  <Button.Icon icon={Booking} large secondary />
</Box>
```

#### With inverted

```javascript
<Box display="flex" justifyContent="space-evenly" width="100%">
  <Button.Icon icon={Booking} inverted/>
  <Button.Icon icon={Booking} inverted secondary />
</Box>
```
