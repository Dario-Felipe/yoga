# Menu

## Reference

Gympass `<Menu />` displays a menu to the user—such as a set of actions or functions—triggered by a button.

- `<Menu.Action />`
- `<Menu.List />`
- `<Menu.Item />`

## Features

- Full keyboard navigation.
- Can be active or disabled Menu.Item.
- Active is managed.
- Icon support on Menu.Item

## Usage

### Menu basic usage

```javascript
    <Box display="flex" flexDirection="column" alignItems="center">
      <Menu>
        <Menu.Action>
          <Button.Icon icon={MenuList} />
        </Menu.Action>
      <Menu.List>
        <Menu.Item>
          one
        </Menu.Item>
        <Menu.Item>
          two
        </Menu.Item>
        <Menu.Item>
          three
        </Menu.Item>
      </Menu.List>  
      </Menu>  
    </Box>
```
### Menu with a icon on Menu.Item

```javascript
    <Box display="flex" flexDirection="column" alignItems="center">
      <Menu>
        <Menu.Action>
          <Button.Icon icon={MenuList} />
        </Menu.Action>
      <Menu.List>
        <Menu.Item icon={FlagBrazil}>
          Brazil
        </Menu.Item>
        <Menu.Item icon={FlagUS}>
          USA
        </Menu.Item>
        <Menu.Item icon={FlagPortugal}>
          Portugal
        </Menu.Item>
      </Menu.List>  
      </Menu>  
    </Box>
```
### Menu on active item and disabled

```javascript
    <Box display="flex" flexDirection="column" alignItems="center">
      <Menu>
        <Menu.Action>
          <Button>Open menu</Button>
        </Menu.Action>
      <Menu.List>
        <Menu.Item active>
          one
        </Menu.Item>
        <Menu.Item>
          two
        </Menu.Item>
        <Menu.Item disabled>
          three
        </Menu.Item>
      </Menu.List>  
      </Menu>  
    </Box>
```
### Menu with href prop

```javascript
    <Box display="flex" flexDirection="column" alignItems="center">
      <Menu>
        <Menu.Action>
          <Button.Icon icon={MenuList} />
        </Menu.Action>
      <Menu.List>
        <Menu.Item href="https://www.google.com" target="blank">
          Google
        </Menu.Item>
        <Menu.Item href="https://www.gympass.com" target="blank">
          Gympass
        </Menu.Item>
        <Menu.Item href="https://gympass.github.io/yoga/" target="blank">
          Yoga
        </Menu.Item>
      </Menu.List>  
      </Menu>  
    </Box>
```
