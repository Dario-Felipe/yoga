# Snackbar

### Reference

Gympass `<Snackbar />` description

### Usage

```javascript state
render(() => {
  const [isOpen, setIsOpen] = useState(false);

  const handleOpen = () => {
    setIsOpen(true);
  };

  const handleOnClose = () => {
    setIsOpen(false);
  };

  const handleOnAction = () => {
    alert('onAction function');
  };

  return (
    <>
      <Button onClick={handleOpen}>Open snackbar</Button>

      <Snackbar
        open={isOpen}
        message="Feedback message."
        actionLabel="Action"
        onAction={handleOnAction}
        onClose={handleOnClose}
      />
    </>
  );
});
```
