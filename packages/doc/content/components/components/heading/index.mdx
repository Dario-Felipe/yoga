### Reference

Gympass `<Heading />` is a component which has the purpose of providing consistent headers

### Usage

#### Default

```javascript
  <Heading>
    <Heading.Title>Gympass</Heading.Title>
  </Heading>
```

#### With back button

```javascript
  <Heading>
    <Heading.BackButton onClick={() => alert('back button')} />
    <Heading.Title>Gympass</Heading.Title>
  </Heading>
```


#### Just back button

```javascript
  <Heading>
    <Heading.BackButton onClick={() => alert('back button')} />
  </Heading>
```

#### With all components

```javascript
  <Heading>
    <Heading.BackButton onClick={() => alert('back button')} />
    <Heading.Title>Gympass</Heading.Title>
    <Heading.RightButton onClick={() => alert('right button 1')} icon={Upload2} />
    <Heading.RightButton onClick={() => alert('right button 2')} icon={Whatsapp} />
  </Heading>
```
