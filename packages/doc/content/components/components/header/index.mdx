# Header

### Reference

The Header displays information and actions relating to the current screen.

### Usage

### Default

```javascript
<Box width="100%">
  <Header />
</Box>
```

### Logo with link

```javascript
<Box width="100%">
  <Header link='/'/>
</Box>
```

### Receiving children

```javascript
<Box width="100%">
  <Header>
    <Button.Text>First step</Button.Text>
    <Button.Text>Second step</Button.Text>
  </Header>
</Box>
```
