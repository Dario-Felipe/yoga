# Result

#### Reference

Gympass `<Result />` component can be used to display content related to a single
subject of on list of searched results for example. The content can consist of multiple elements of varying types and
sizes.

As it could be seen in most components, the Result has its own compounds:

- `<Result.Details />`
- `<Result.Tags />`
- `<Result.Button />`

Each one of the compounds has its own props described on the example below.
