## Usage

### Individual

Icon components receive all the available svg tag attributes.

```javascript
<Star stroke="#ffc24c" width={26} height={26} />
```

### Group

You can use the icon group of `FlagsIcons` or `UIIcons`.

Remembering that icon groups are `objects` you can work with.

```javascript
<Box d="flex" gap="small">
  {Object.entries(FlagsIcons).map(([_, Component]) => (
    <Component width={26} height={26} />
  ))}
</Box>
```

You also can use a specific icon from a group.


```javascript
<UIIcons.Favorite fill="#d8385e" width={26} height={26} />
```
